# r00t-apk.github.io

Personal site built from Bruno Simon's open-source [`folio-2025`](https://github.com/brunosimon/folio-2025)
portfolio (an interactive Three.js / WebGL experience), released under the MIT License.

The site is built with Vite and deployed automatically to GitHub Pages via
[`.github/workflows/deploy.yml`](.github/workflows/deploy.yml) on every push to `main`.

## Local development

```bash
cp .env.example .env
npm install --force
npm run dev     # serve locally
npm run build   # build to dist/
```
